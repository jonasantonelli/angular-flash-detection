Usage

Via attributes, the dropdown box directive toggles hidden content by toggling the .in class on the parent item.

Add dropdown-box attribute to a link or a button.


Setting the content target

Use the attribute target="content" as value of the #id of the content referenced in the dropdown-box

Directions

For setting direction, use placement="value" and one of following values left right top bottom

Disable arrow

If you want disable the arrow of container that will toggle adding attribute arrow="false".

Mouseover Event

You can allow use of the event mouseover adding attribute mouseover

Delay

You can use delay in milliseconds for events mouseover adding attribute delay="4000" or only delay (default=2000)
